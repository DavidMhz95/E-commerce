eMarket Services 
=======================

Prerequisites
-------------

For work with this project, you need the follow:

1. This project only works with Visual Studio 2017 or above
2. You need install NodeJS (https://nodejs.org/es/)

How to compile
--------------

If you have problems with the compilation related with sass module, try to execute this on the root of the project:
* npm rebuild node-sass
Rebuild again and it's work!
